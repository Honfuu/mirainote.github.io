よくある質問

お客様からよくいただくご質問と回答をご紹介いたします

①帰りも電話する必要あるの？

+

往復、付き添い全て対応するので、帰りの心配は要りません。（生活支援サービスの場合）

②65歳以下だけど利用できる？

+

生活支援サービスに関しては、年齢制限ありません。どなたでもご利用いただけます。介護タクシーは、原則65歳以上、または妊婦、障がい者の方。

③タクシーとの違いは？

+

タクシーとの違いは、完全予約制。福祉専門の有資格者が対応するところです。

④1時間というのは、どこからスタート？

+

弊社の生活支援サービスでは、目的地の滞在時間のみ時間に含めています。移動時間は、含めていないので、ご安心ください。

⑤外出以外にも対応してくれるの？ゴミ出しとか掃除とか

+

対応可能です。事前にご相談いただければ対応いたします。

⑥定期利用も可能？

+

可能です。事前にご相談し、日程調整いたします。

⑦介護保険使える？

+

弊社のサービスは、全て自費になっております。その代わり、介護保険の枠組みにとらわれない、柔軟なサービス提供することが可能です。

⑧どれくらい前に予約すればいいの？

+

当日予約も可能ですが、予定が埋まっている場合もありますので、できるだけ前日までにご連絡いただけると確実です。

⑨急な予定変更や延長は可能？

+

可能な限り対応いたしますが、後ろの予定がある場合はお受けできないこともあります。できるだけ早めにご相談ください。

⑩同居家族の代わりに予約してもいい？

+

ご家族やケアマネジャーの方からのご予約・ご相談も承っております。安心してご依頼ください。

⑪車いすに乗ったままでも利用できるの？

+

はい、介護タクシーでは車いすのまま乗車できる専用車両で対応いたします。生活支援サービスでも必要に応じて対応いたしますのでご相談ください。

⑫利用できる時間帯は？夜や早朝は？

+

基本は【8時30分～18時】ですが、早朝・夜間対応をご希望の方は、事前にご相談ください。

⑭サービス当日、何を準備しておけばいいの？

+

特別な準備は必要ありません。外出や付き添い内容に応じて、必要なもの（診察券・お薬手帳など）をお持ちください。

他にもご質問がございましたら、お気軽にお問い合わせください。
